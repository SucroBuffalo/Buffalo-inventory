📚 Setup Guide
1

Google Cloud Project

Enable Sheets API + Drive API. Create OAuth 2.0 credentials (Web app). Add your site URL to Authorized Origins. Copy Client ID & API Key above.

2

Create Google Spreadsheet

New Google Sheet with these tabs: Inbound LiquidSugar TruckOut RailOut Molasses DailySummary. Copy the Sheet ID from the URL.

3

EmailJS Setup

Sign up at emailjs.com · Connect Gmail · Create template with: {{date}} {{loss_pct}} {{inbound}} {{outbound}}. Set recipient to buffalo@sucro.us
